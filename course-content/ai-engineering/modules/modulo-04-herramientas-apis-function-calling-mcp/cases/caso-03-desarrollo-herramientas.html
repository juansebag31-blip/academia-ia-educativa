Desarrollo: tool calling seguro en un repositorio

Un copiloto debe inspeccionar código, ejecutar pruebas y preparar un commit sin publicar cambios automáticamente.

Flujo

Leer archivos→Modificar→Ejecutar tests→Mostrar diff→Aprobación→Commit

Controles

  • Namespace limitado al repositorio activo.
  • Timeout para comandos.
  • Lista de operaciones permitidas.
  • Prohibición de secretos en contexto.
  • Traza de cada ejecución.
  • Confirmación antes de push o deploy.
