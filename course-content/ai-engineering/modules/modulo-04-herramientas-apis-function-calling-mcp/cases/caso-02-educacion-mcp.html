Educación: recursos y herramientas mediante MCP

Una plataforma educativa conecta un repositorio de contenidos con distintos asistentes de IA.

Primitivas

PrimitivaUso
ResourceLeer programa, materiales y rúbricas.
PromptAplicar una plantilla de retroalimentación.
ToolRegistrar una devolución o crear una actividad.

Decisión arquitectónica

MCP permite reutilizar el mismo servidor con varios hosts, mientras que una integración directa sería más simple para una única aplicación cerrada.

AprendizajeEl valor de MCP aparece cuando la interoperabilidad y el descubrimiento compensan su complejidad inicial.
