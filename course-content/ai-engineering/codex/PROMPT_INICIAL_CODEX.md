# Prompt inicial para Codex

Trabaja sobre el repositorio de mi curso existente Academia IA Educativa. Antes de cambiar código, inspecciona su estructura, tecnologías, sistema de rutas, componentes reutilizables, gestión de contenido, estilos y despliegue. Toma `https://academia-ia-educativa.vercel.app/visual` como referencia funcional del producto actual.

He añadido la carpeta `course-content/ai-engineering/`, que contiene el Módulo 1 aprobado del nuevo curso **AI Engineering Aplicado**. Lee primero:

1. `AGENTS.md`
2. `README.md`
3. `module-manifest.json`
4. Los archivos HTML y assets declarados en el manifiesto.

Objetivo de esta primera tarea:

- No construyas todavía los doce módulos.
- No crees todavía una skill definitiva.
- Diseña e integra únicamente el Módulo 1 como prototipo fundacional.
- Reutiliza la arquitectura del curso actual cuando sea conveniente, pero crea una identidad visual propia para AI Engineering.
- Separa contenido y componentes para que luego podamos repetir el patrón.

Primero entrégame:

1. diagnóstico de la arquitectura actual;
2. propuesta de estructura de rutas y componentes;
3. mapeo de cada archivo aprobado a la interfaz;
4. riesgos y decisiones técnicas;
5. plan incremental en pequeñas tareas.

Espera mi aprobación del plan antes de realizar una refactorización grande.

Después de la aprobación, implementa una página de módulo con estas secciones:

- cabecera y objetivos;
- cuadernillo fundacional;
- experiencia visual y auditiva;
- casos reales;
- presentación descargable;
- actividad y autoevaluación;
- fuentes y progreso.

Conserva íntegramente el contenido aprobado. Ejecuta build y pruebas. Al final informa archivos modificados, decisiones tomadas y pasos pendientes.
